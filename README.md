> 必要なため**保留中**です。ビルド・配布にご協力いただける方を募集しています。

スマートフォン単独（ネットにつながないで動作）で使える音声通訳アプリ。
端末内の生成AI **Gemma 3n（E4B）** が音声認識と翻訳を行い、内蔵音声合成で読み上げます。
Google Pixel 9a 以上を想定。

> **⏳ 本バージョンは試用版です**：**2026-07-31 まで無料・全機能（医学用語RAG含む）が利用可能**。
> 期限後はアプリの利用に**解除コード**が必要になります（本バージョンは広告なし）。
> 端末ごとの解除コードは提供元へアクティベーションIDを送ってお求めください。

```
マイク(本体/USB) → 録音 → Gemma 3n E4B(音声→原文＋翻訳) → 端末内TTSで発声
```

## 特長
- **STT＋翻訳をオンデバイス**：Gemma 3n E4B（音声入力対応）。通信不要・オフライン。
- **TTS**：端末内蔵の音声合成（Android=TextToSpeech / iPhone=AVSpeechSynthesizer。オフライン可）。
- **2マイク対応**：言語1・言語2で共通の本体マイクを使うことも、USBマイクをつけて
  言語1・言語2をマイクごとに分けて認識させることも可能。
- **言語**：言語1＝端末の設定言語（既定）、言語2＝選択した外国語（全17言語）。押したボタンの言語で認識。
- **録音方式**：押して話す／タップ切替（タップ時は無音で自動確定・感度は自動/手動）。
- **医学用語オプション（RAG）**：眼科を中心とした医療用語の精度を向上（暗号化パック・端末バインド解錠）。
- **UI**：言語1=右／言語2=左のチャット、文字サイズ、発声ON/OFF、印刷（全て/言語1/言語2）。UI言語は端末設定に追従。

## 動作環境
- Android 12 以上（arm64）。iPhone は別途 iOS 版（TestFlight 配布予定）。
- 翻訳モデル（約4.9GB）は初回にアプリ内のボタンでダウンロード（Wi-Fi推奨・レジューム対応）。

## インストールと使い方（Android）
1. 最新リリースから APK をダウンロードしてインストール（「提供元不明アプリ」の許可が必要）。
2. アプリを開き、ヘッダの「⬇ モデルDL」でモデルを取得（進捗・残り時間を表示）。
3. 「準備完了」後、言語1／言語2 ボタンで話すと翻訳されます。

## 翻訳モデルのダウンロード（GitHub 一本化）
- モデルは本リポジトリの **Releases** に分割アップロード（GitHubの2GB/ファイル制限のため約1.8GB×3に分割）。
- アプリは Release の **`manifest.json`** を読み、分割パートを自動でダウンロード・結合します（既定でGitHubを参照）。
- 保存先はアプリ専用領域のため、**アンインストールでモデルも自動削除**。
- ダウンロード元はアプリ設定で変更可（GitHub以外のサーバや Hugging Face も指定可）。

## 翻訳モデルとライセンス（重要）
本アプリは Google の **Gemma 3n** を利用します。利用は
[Gemma 利用規約](https://ai.google.dev/gemma/terms) と
[使用禁止ポリシー](https://ai.google.dev/gemma/prohibited_use_policy) に従います。
配布物に `NOTICE` を同梱。オンデバイス推論は Google AI Edge LiteRT / LiteRT-LM を使用。

## 医学用語（RAG）
医療・眼科の用語精度を高める機能を内蔵（試用期間中は有効）。暗号化パックを同梱し、
期限後は端末バインドの解除コードで継続利用できます（提供元：Batake）。
